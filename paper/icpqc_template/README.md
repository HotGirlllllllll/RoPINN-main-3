# ICPQC Template Notes

Source page: https://www.ic-pqc.com/download  
Downloaded file: `Template.zip` (official site download item: `Template.zip`, created 2025-11-04)

The official package currently provides a Word template (`ProcSPIETemplate_A4.docx`) and checklist PDFs, not a LaTeX class.

## Implemented in `paper/main.tex`

- A4 paper size
- Margins: top `2.54cm`, bottom `4.94cm`, left/right `1.93cm`
- Single-column layout
- No page numbers / headers / footers
- Times-style text (`times` package)
- SPIE-like title/authors/abstract/keywords block
- Numbered sections with centered section headings
- Numbered references at end

## Manual fields to replace before submission

- `Author One`, `Author Two`
- affiliations and postal code
- corresponding author email
